# ReadMe Maker

## Description
This app has been made to quickly help create professional readmes!

## Installation
Please run the commands 

``` bash
npm init
```
 and then 

 ``` bash
 npm i inquirer
 ```

## Usage
After the installations, you begin the code by running the command 'node index.js' and then answer the questions with the content you would like to fill each section of a professional readme. Afterwards, the app will create a readme with all the information you provided!
